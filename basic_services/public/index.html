Express

Az express modul néhány alapszolgáltatását bemutató példák

Az expressen kívül az alábbi modulok is használatban lesznek:

  • body-parser – JSON, Raw, Text és URL kódolású űrlapadatok kezelésére npm install body-parser --save
  • cookie-parser - A Cookie fejlécének elemzése és a req.cookie-k feltöltése a cookie-nevek által kulcsolt objektummal npm install cookie-parser --save
  • multer - többrészes/form-adatok kezelésére npm install multer --save
  • GET
  • POST
