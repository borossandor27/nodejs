Az express modul néhány alapszolgáltatását bemutató példa
Az expressen kívül az alábbi modulok is használatban lesznek:
 - body-parser – Ez egy node.js köztes szoftver JSON, Raw, Text és URL kódolású űrlapadatok kezelésére.
 - cookie-parser − A Cookie fejlécének elemzése és a req.cookie-k feltöltése a cookie-nevek által kulcsolt objektummal.
 - multer – többrészes/form-adatok kezelésére.

$ npm install body-parser --save
$ npm install cookie-parser --save
$ npm install multer --save
